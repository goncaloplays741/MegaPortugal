# Mega Portugal

Addon para Kodi com canais de televisão portugueses.

## Funcionalidades

- Sport TV 1-6+
- Canais Portugal (RTP, SIC, TVI, CNN Portugal)
- EPG incluído
- Proxy nativo com fallback direto

## Instalação

1. Descarregar o ZIP da [última release](https://github.com/goncaloplays741/MegaPortugal/releases)
2. Kodi → Addons → ⋮ → Install from zip file
3. Selecionar o ficheiro ZIP
